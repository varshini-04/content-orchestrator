# 🤖 Multi-Agent Content Orchestrator

A high-performance AI system built with **Llama 3.3** (via Groq) that utilizes a multi-agent architecture to research, draft, and refine technical content.

🚀 **[Live Demo](https://content-orchestrator-nnsdasrwqcjnehvvuhyqet.streamlit.app/)**

## 🛠️ Tech Stack
- **LLM:** Llama 3.3 (Groq API)
- **Framework:** Python, Streamlit
- **Architecture:** Multi-Agent (Research, Content Creator, Critic, Adapter)
- **Validation:** Pydantic for structured outputs

## 🧠 Features
- **Self-Reflection Loops:** The Critic agent reviews content to ensure high technical accuracy.
- **Structured Workflows:** Uses ReAct-style logic to orchestrate complex writing tasks.
- **Modern UI:** Clean Streamlit interface for seamless user interaction.
